## 🚀 After Adding Files:

1. **Navigate to project folder:**
   ```bash
   cd maple-flow-bus-tracker
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the app:**
   ```bash
   npm start
   ```

4. **Open browser to** `http://localhost:3000`
